API Project: Request Header Parser Microservice

User Story:

1. To get the IP address, preferred languages (from header Accept-Language)
and system infos (from header User-Agent) for client's device.

Example Usage:

[base url]/api/whoami/check

Example Output:

{"ipaddress":"::ffff:159.20.14.100",
"language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
