API Project: URL Shortener Microservice

User Stories:

  1. POST a URL to [project_url]/api/shorturl/new and receive a shortened URL which could be copied
  2. Passing an invalid URL that doesn't follow the http(s)://www.example.com
    (/more/routes)     format, the JSON response will contain an error like {"error":"invalid URL"}
  3. When visiting the shortened URL, it will redirect to the original link.

Short URL Creation

example: POST [project_url]/api/shorturl/new - https://www.google.com

Example Usage:

[this_project_url]/sh/7HKaSo
